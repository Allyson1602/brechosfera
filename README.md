# Brechosfera Frontend

## Setup local

Requisitos:
- Node.js 18+
- npm ou pnpm

Instalar dependencias:

```sh
npm install
```

## Variaveis de ambiente

Desenvolvimento:

```sh
VITE_API_BASE_URL=http://localhost:3000
```

Producao:

```sh
VITE_API_BASE_URL=https://api-brechosfera.vercel.app
```

## Scripts

Rodar em desenvolvimento:

```sh
npm run dev
```

Build de producao:

```sh
npm run build
```

Preview local do build:

```sh
npm run preview
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui
