# Brechosfera Frontend

## Setup local

Requisitos:
- Node.js 18+
- npm ou pnpm

Instalar dependências:

```sh
npm install
```

Rodar em desenvolvimento:

```sh
npm run dev
```

Build de produção:

```sh
npm run build
```

Preview local do build:

```sh
npm run preview
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui
